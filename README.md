# willmarella.github.io

Personal portfolio site built with Jekyll, hosted on GitHub Pages.

## Setup

1. Create a repo named `willmarella.github.io` on GitHub
2. Clone it and copy these files in
3. Add your headshot as `assets/images/headshot.jpg`
4. Add your CV as `assets/WillMarella_CV.pdf`
5. Push to `main` — GitHub Pages builds and deploys automatically

## Local preview (optional)

```bash
gem install bundler
bundle install
bundle exec jekyll serve
```

Then open http://localhost:4000
